## 🚀 Cara Menjalankan

```bash

cd -ML-Diamond-Generator

python3 -m http.server 8080
```

Repository ini siap di-deploy ke GitHub Pages.

1. Buka repository di GitHub
2. Pilih **Settings** > **Pages**
3. Pilih branch **main** dan folder **/ (root)**
4. Klik **Save**
